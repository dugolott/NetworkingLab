## 2. Wireshark

Strumento di analisi del traffico di rete, permette di catturare e analizzare i pacchetti che transitano sulla rete.

Usato nel mondo accademico e professionale per

- il troubleshooting networking e software development
- cybersecurity - analisi malware, pentesting, red teaming
- studio protocolli di rete
- analisi forense
- analisi performance di rete
- ...

### Concetti chiave

1. Interfaccia di cattura
2. Aree di lavoro
   2.a Lista pacchetti / Colonne
    2.b Dettagli pacchetto
    2.c Payload
3. Filtri
  3.a Display filter
  3.b Capture filter
4. Statistiche:
  4.a Protocol hierarchy
  4.b Conversations
  4.c Endpoint
5. Analisi dei pacchetti
  5.a esportazione payload
  5.b Ricostruzione flussi

Nota - Per la challenge Network 07 - Stream TCP, possiamo utilizzare Wireshark per catturare il traffico e poi esportare i dati del payload TCP in un file. Successivamente, possiamo utilizzare Python per analizzare il file e ricostruire il messaggio nascosto.

#### Esempio di estrazione del payload TCP con Scapy

```python

from scapy.all import rdpcap, TCP, Raw
pkts = rdpcap("c:/..../Download/capture.pcap")
f = b""
for p in pkts:
     if TCP in p and Raw in p:
         f+=p[Raw].load.replace(b"\n", b"").replace(b"\r",b"")


```
