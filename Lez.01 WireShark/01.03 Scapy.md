# Analisi del traffico di rete con Scapy

In parecchi casi è necessario analizzare il traffico di rete a livello di pacchetto. Scapy è una libreria Python che permette di creare, inviare, ricevere e analizzare pacchetti di rete.


```bash

pip install scapy

```

Possiamo utilizzare Scapy per leggere un file PCAP e analizzare i pacchetti DNS al suo interno. Ad esempio, se abbiamo un file `capture.pcap` che contiene traffico di rete, possiamo estrarre tutte le query DNS in questo modo:


```python
from scapy.all import *
packets = rdpcap('capture.pcap')
for p in packets:
    if p.haslayer('DNS'):
        print(p['DNS'].qd.qname)

```


Facciamo un passo oltre e affrontiamo una challenge un po' più complessa.

**Dispositivo Non Sicuro**  --> https://training.olicyber.it/challenges#challenge-143

La challenge fornisce anche il link alla writeup, che si basa sull'utilizzo del tool CLI tshrk. Nel seguito propongo la soluzione con script python  in abbinamento alla libreria scapy.

```python
from scapy.all import rdpcap, DNS, DNSQR

chunks = {}
file_pcap = "capture.pcapng"
file_out = "output"

for p in rdpcap(file_pcap):
    if not (p.haslayer(DNS) and p[DNS].qr == 0 and p.haslayer(DNSQR)):
        continue

    qname = p[DNSQR].qname
    if isinstance(qname, bytes):
        qname = qname.decode(errors="ignore")
    qname = qname.rstrip(".")

    if "attacker.eve" not in qname:
        continue

    n, data, *_ = qname.split(".")
    if n.isdigit():
        chunks[int(n)] = data

with open(file_out, "wb") as f:
    for i in sorted(chunks):
        f.write(bytes.fromhex(chunks[i]))
```
