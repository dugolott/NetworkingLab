# Analisi del traffico di rete con PySHark

In realtà la libreria più indicata per l'analisi di pcap voluminosi è PyShark. Come Wireshhark, si basa sul formato dello strim esposto da Winpkap/libpcap, pertanto anche sintassi e filtri risultano familiari a chi conosce questo programma.

A differnza di scapy, pyshark non carica tutto il pcap in RAM, ma mantiene aperto il file durante l'analisi. I pacchetti vengono caricati a mano a mano, durante il parsing del ciclo di scorrimento.

Consideriamo la stessa challenge

- Dispositivo Non Sicuro  --> https://training.olicyber.it/challenges#challenge-143


```python
import pyshark

chunks = {}
cap = pyshark.FileCapture("capture.pcapng", display_filter="dns")

for pkt in cap:
    try:
        if pkt.dns.flags_response != '0':
            continue

        qname = pkt.dns.qry_name.rstrip(".")

        if "attacker.eve" not in qname:
            continue

        n, data, *_ = qname.split(".")
        if n.isdigit():
            chunks[int(n)] = data

    except AttributeError:
        continue

cap.close()

with open("output", "wb") as f:
    f.write(bytes.fromhex("".join(chunks[i] for i in sorted(chunks))))
```
